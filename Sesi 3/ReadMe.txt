-----Fundamental Programming with Looping-----

Looping Introduction
- Loop adalah sebuah hal yang digunakan untuk mengeksekusi sebuah set dari suatuinstruksi atau fungsi secara berulang ketika beberapa kondisi menjadi nyata (true).

--2 tipe loop pada Java -> for dan while -> do while--
- for adalah pernyataan aliran kontrol yang mengulang bagian dari program beberapakali
- while adalah pernyataan aliran kontrol yang mengeksekusi bagian dari programberulang kali berdasarkan kondisi boolean yang diberikan.
- do-while adalah pernyataan aliran kontrol yang mengeksekusi bagian dari programsetidaknya satu kali dan eksekusi selanjutnya bergantung pada kondisi boolean yang diberikan.

Waktu Penggunaan
- jumlah iterasi tetap -> for ex. for (int i - 0; i < 2; i++) {}
- jumlah iterasi tdk tetap -> while
- jumlah iterasi tdk tetap dan harus loop sekali -> do-while

--2 jenis loop--
1. counted loop -> for dan for-each (untuk menampilkan isi array)
2. uncounted loop -> do dan do-while

--Break & Continue--
Break membuat loop berhenti dan beralih ke pernyataan berikutnya
‘continue’ akan menghentikan iterasi saat ini dan pindah ke berikutnya
