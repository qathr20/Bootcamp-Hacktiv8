------Setting Java PATH di Windows------
1. Buka directori dari java yang telah anda install
2. Cari 'Environment' pada menu search di Windows
3. Klik 'Environment Variables'
4. Klik 'Edit' pada bagian System Variables
5. Kemudian klik New, dan paste path yang telah anda copy di tahap 3, kemudian klik 'OK
6. Cari CMD pada menu search di windows anda kemudian 'Run as administrator'
7. Tulis 'javac' atau 'java' pada CMD

- Package adalah  folder yang berisi sekumpulan program Java.
- Library merupakan sekumpulan class dan fungsi dalam membuat program.
- Setiap program harus dibungkus di dalam class untuk nanti dibuat menjadi objek
- Method main() atau fungsi main() merupakan blok program yang akan SELALU dieksekusi pertama kali
- Statement adalah unit sintaks pada bahasa pemrograman yang menyatakan aksi atau tugas untuk dilakukan.
- Ekspresi adalah entitas sintaks pada bahasa pemrograman yang menghasilkan sebuah nilai
- Statement dan ekspresi adalah bagian terkecil dalam program yang  harus diakhiri dengan titik-koma ( ; )
-  Input : Nilai yang dimasukkan dalam program
- Program : Langkah demi Langkah yang dilakukan untuk mengelola input
- Output : Hasil

-----Tipe Data-----
char : tipe data karakter, ex Y
int : angka atau bilangan bulat, ex 29
float : bilangan desimal, x 2.1
double : bilangan desimal juga, tapi lebih besar kapasitasnya, ex 2.1
string : kumpulan dari karakter yang membentuk teks, ex Hello Java!
Boolean : tipe data yang hanya bernilai true dan false
